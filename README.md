# 📝 Task Manager CLI

A simple terminal-based Task Manager built in Go. Supports basic CRUD operations to manage your daily tasks using an in-memory store.


---

## 🚀 Features

- Create a task
- View all tasks
- Mark tasks as done
- Delete tasks
- Interactive terminal menu using `bufio`

---

## 🛠 Requirements

- Go 1.18+

---

## ▶️ Running the App

```bash
go run main.go
